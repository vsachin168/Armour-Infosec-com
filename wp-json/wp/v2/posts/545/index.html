RPM Packages<\/a><\/strong><\/p>\n

RPM stands for Red Hat Package Manager. RPM is a powerful Package Manager for Red Hat, Suse and Fedora Linux. It can be used to build, install, query, verify, update, and remove\/erase individual software packages. A Package consists of an archive of files, and package information, including name, version, and description.<\/p>\n

    \n
  • Files end in the .rpm extension<\/li>\n
  • RPM files are the preferred Linux installation files according to the LSB (Linux Standard Base)<\/li>\n
  • Two different types of RPM files:\n
      \n
    • RPM – contains the compiled code<\/li>\n
    • SRPM – contains the source code for a particular application<\/li>\n<\/ul>\n<\/li>\n
    • RPM package names are constructed like this:<\/li>\n<\/ul>\n

      name-version-release.architecture.rpm<\/p>\n

      openssh-5.3p1-52.el6_1.2.x86_64.rpm<\/p>\n

        \n
      • Architecture comes in three types:\n
          \n
        • I386 for 32 bit system<\/li>\n
        • x86_64 for 64 bit systems<\/li>\n
        • noarch – platform independent software<\/li>\n<\/ul>\n<\/li>\n
        • RPM packages contain the following items:\n
            \n
          • SPEC file – A file that ends in the .spec file extension, contains the information necessary to create the RPM package<\/li>\n
          • The application or libraries that are to be installed<\/li>\n
          • Any documentation (such as man pages ) that come with the package<\/li>\n
          • Any configuration files that come with the application<\/li>\n<\/ul>\n<\/li>\n
          • Package installation is stored in the RPM database. RPM database is located at: -\/var\/lib\/rpm<\/li>\n<\/ul>\n

            RPM Commands<\/h4>\n

            list of all installed packages<\/p>\n

            # rpm -qa<\/p>\n

            list of all recently installed packages<\/p>\n

            # rpm -qa –last<\/p>\n

            Query a particular Package<\/p>\n

            # rpm -qa packagename<\/p>\n

            Information about Package<\/p>\n

            # rpm -qi packagename<\/p>\n

            Documentation files of Package<\/p>\n

            # rpm -qd packagename<\/p>\n

            Dependencies a rpm file has<\/p>\n

            # rpm -qR packagename<\/p>\n

            list of configuration file for a Package<\/p>\n

            # rpm -qc packagename<\/p>\n

            list of all the Files in a Package<\/p>\n

            # rpm -qpl packagename.rpm<\/p>\n

            To Install the package<\/p>\n

            # rpm -ivh packagename.rpm<\/p>\n

            To Upgrade package<\/p>\n

            # rpm -Uvh packagename.rpm<\/p>\n

            Forcefully install package<\/p>\n

            # rpm -ivh –nodeps packagename.rpm<\/p>\n

            Uninstall package<\/p>\n

            # rpm -e packagename<\/p>\n

            Forcefully uninstall package<\/p>\n

            # rpm -e –nodeps package<\/p>\n

            Erase\/remove\/ an installed package<\/p>\n

            # rpm -ev package<\/p>\n

            Erase\/remove\/ an installed packagewithout checking for dependencies<\/p>\n

            # rpm -ev –nodeps package<\/p>\n","protected":false},"excerpt":{"rendered":"
