[vc_row][vc_column][vc_column_text]Before you lose or forgot your password and left with frustration and exasperation, you should know how to change or reset it. So here\u2019s our post regarding the cure of most common problem arisen.<\/p>\n

Resetting password of RHEL, CentOS and Fedora :-
\n1.\u00a0\u00a0 \u00a0Start your computer and wait for the boot menu to appear.<\/p>\n

2.\u00a0\u00a0 \u00a0At the grub boot menu, press e to edit the existing kernel (Core).<\/p>\n

3.\u00a0\u00a0 \u00a0To modify it just change read only mode (ro) to\u00a0rw\u00a0(write mode) and add init=\/sysroot\/sbin\/sh\u00a0like below screenshot then\u00a0press\u00a0F10 or CTRL+X\u00a0to reboot the Kali Linux.<\/p>\n

4.\u00a0\u00a0 \u00a0In the single user mode, run the command > “chroot\u00a0 \/sysroot”.<\/p>\n

5.\u00a0\u00a0 \u00a0Finally, run the commands \u201cpasswd root” You\u2019ll be prompted to create and confirm a new password.<\/p>\n

6.\u00a0\u00a0 \u00a0After creating the password, run the commands below to update SELinux parameters\u00a0 “touch \/.autorelabel”<\/p>\n

7.\u00a0\u00a0 \u00a0reboot -f
