<\/a>Top hacking apps for Android smart phones and tablets<\/h4>\n

AnDOSid<\/strong><\/p>\n

AnDOSid tag\u2019s posts with two unique numbers which relate to the Android device that sent the request. AnDOSid allows security professionals to simulate a DOS attack.<\/p>\n

AndroRAT<\/strong><\/p>\n

Androrat<\/strong> is a client\/server application developed in Java Android for the client side and in Java\/Swing for the Server. The name Androrat<\/strong> is a mix of Android and RAT (Remote Access Tool)<\/p>\n

APKInspector<\/strong><\/p>\n

APKinspector is another open source project that comes to reverse and analyze Android applications. project owners have created a graphical interface to allow visualizing the structure of the application modules this will make security analysts select the good Android application that is safe to use.<\/p>\n

dSploit<\/strong><\/p>\n

dSploit is an advance toolkit to perform network security assesments on mobile phones. It is a complete toolkit so you can perform various attacks like password sniffing, real time traffic manipulation, etc<\/p>\n

DroidSheep<\/strong><\/p>\n

DroidSheep [Root] is an Android app for Security analysis in wireless networks and capturing facebook, twitter, linkedin and other accounts.<\/p>\n

Faceniff<\/strong><\/p>\n

FaceNiff is an Android app that allows you to sniff and intercept web session profiles over the WiFi that your mobile is connected to.<\/p>\n

Hackode <\/strong><\/p>\n

Hackode is an android app developed by Ravi Kumar for penetration testers, ethical hackers and cyber security professionals. It contains various modules including Reconnaissance, Scanning, Exploits and Security Feed.<\/p>\n

Nmap for Android<\/strong><\/p>\n

Nmap is a popular network security scanner which is also available for android devices. It is used by professionals for network exploration. It works on both non-rooted and rooted phones, But if your device is rooted then you have access to some more features.<\/p>\n

Penetrate Pro<\/strong><\/p>\n

Penetrate Pro is an excellent Android app that calculates WEP \/ WPA keys for some wireless routers. Also, no advertising and 3G routers search based on Thomson.<\/p>\n

\u00a0SpoofApp<\/strong><\/p>\n

SpoofApp<\/em> is a Revolutionary iPhone Caller ID Spoofing, Voice Changing, and Call Recording Application.<\/p>\n

Network Spoofer<\/strong><\/p>\n

Network Spoofer lets you change websites on other people’s computers from an Android phone.<\/p>\n

WhatsApp Sniffer<\/strong><\/p>\n

WhatsApp Sniffer Android APK is a Android Application that lets you see anyone\u2019s WhatsApp Conversations as long as you know their phone number<\/p>\n

\u00a0Wi-Fi Killer<\/strong><\/p>\n

this app you can disable internet connection for a device on the same network.
